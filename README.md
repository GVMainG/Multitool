# Multitool - Модульный WPF мультитул на .NET 9

Приложение с модульной архитектурой и навигацией между инструментами через систему страниц.

## Структура проекта

```
Multitool/
├── Multitool.Core/           # Интерфейсы и модели
├── Multitool.Infrastructure/ # Реализация бизнес-логики
├── Multitool.UI/             # WPF приложение
│   ├── Models/               # Модели данных
│   ├── Views/                # Страницы (Pages)
│   ├── ViewModels/           # ViewModel для страниц
│   ├── Services/             # Сервисы (навигация и др.)
│   └── App.xaml.cs           # Настройка DI контейнера
└── Multitool.sln             # Решение
```

## Требования

- **.NET 9 SDK** (https://dotnet.microsoft.com/download/dotnet/9.0)

## Установка и запуск

### 1. Восстановление пакетов
```bash
dotnet restore
```

### 2. Сборка проекта
```bash
dotnet build --configuration Release
```

### 3. Запуск приложения
```bash
dotnet run --project Multitool.UI
```

## Как использовать

### Главный экран
При запуске отображается главное меню с плитками инструментов. Каждая плитка содержит:
- **Иконку** — визуальное представление инструмента
- **Название** — имя инструмента
- **Описание** — краткое описание функциональности

### Навигация
- **Выбор инструмента** — кликните на плитку для перехода к инструменту
- **Возврат назад** — кнопка "← Назад" в верхней части страницы инструмента
- **Главное меню** — кнопка возврата всегда возвращает на главную страницу

### Доступные инструменты

#### 🎵 Извлечение аудио
Извлечение аудиодорожки из видеофайлов.

1. Нажмите на плитку **"Извлечение аудио"**
2. Нажмите **"Обзор..."** для выбора видеофайла
3. Путь сохранения заполнится автоматически (можно изменить через второй "Обзор...")
4. Нажмите **"Извлечь аудио"**
5. Дождитесь завершения процесса

**Поддерживаемые форматы:**
- Входные: MP4, AVI, MKV, MOV, WMV, FLV
- Выходной: MP3

## Настройка FFmpeg

Приложение **автоматически загружает FFmpeg** при первом использовании:
- Загружается официальная сборка с gyan.dev
- Сохраняется в `%LOCALAPPDATA%\Multitool\ffmpeg\`
- Повторно используется без загрузки

**Примечание:** Требуется подключение к интернету для первой загрузки (~100 МБ).

## Расширение функциональности

### Добавление нового инструмента

1. **Создайте страницу инструмента** в `Multitool.UI/Views/`:
   ```xml
   <!-- MyToolPage.xaml -->
   <Page x:Class="Multitool.UI.Views.MyToolPage" ...>
       <!-- UI инструмента -->
   </Page>
   ```

2. **Создайте ViewModel** в `Multitool.UI/ViewModels/`:
   ```csharp
   public class MyToolViewModel : ObservableObject
   {
       // Логика инструмента
   }
   ```

3. **Зарегистрируйте инструмент** в `MainMenuViewModel.cs`:
   ```csharp
   Tools = new ObservableCollection<ToolDescriptor>
   {
       new ToolDescriptor
       {
           Id = "my-tool",
           Name = "Мой инструмент",
           Description = "Описание",
           Icon = "🔧",
           PageType = typeof(Views.MyToolPage)
       }
   };
   ```

4. **Добавьте зависимости** в `App.xaml.cs`:
   ```csharp
   services.AddTransient<MyToolViewModel>();
   services.AddTransient<MyToolPage>();
   ```

5. **Реализуйте логику** в `Multitool.Infrastructure/` при необходимости.

## Архитектура

- **Dependency Injection** через Microsoft.Extensions.Hosting
- **MVVM** с CommunityToolkit.Mvvm
- **Навигация** через Frame и INavigationService
- **Асинхронные операции** с поддержкой отмены
- **Слабая связанность** через интерфейсы

## Компоненты

### Сервисы
- **INavigationService** — управление навигацией между страницами

### Модели
- **ToolDescriptor** — описание инструмента для отображения в меню

### Страницы
- **MainMenuPage** — главное меню с плитками инструментов
- **AudioExtractorPage** — страница извлечения аудио

### ViewModel
- **MainMenuViewModel** — управление главным меню
- **AudioExtractorViewModel** — логика извлечения аудио
